
# TallerPython_BarronGonzalezJoseAngel

Entrega del taller **"Python para la Ciencia, Tecnología e Ingeniería"**.

## Descripción del proyecto
Este repositorio contiene los ejercicios desarrollados durante el taller, enfocados en los fundamentos de programación en Python aplicados a contextos científicos e ingenieriles. Incluye manejo básico de entrada/salida de datos, operaciones aritméticas, declaración de variables y uso introductorio de librerías como `numpy`, `pandas` y `matplotlib`.

## Instrucciones para clonar el repositorio
```bash
git clone https://github.com/Lnexotic/TallerPython_BarronGonzalezJoseAngel.git
cd TallerPython_BarronGonzalezJoseAngel
```

## Cómo instalar dependencias
Instala las librerías necesarias ejecutando:
```bash
pip install -r requirements.txt
```

## Cómo ejecutar el código
El cuaderno principal se encuentra en la carpeta `notebooks/` con el nombre:  
`TallerPython_BarronGonzalezJoseAngel.ipynb`

Puedes abrirlo con:
- **Jupyter Notebook**:
  ```bash
  jupyter notebook notebooks/TallerPython_BarronGonzalezJoseAngel.ipynb
  ```
- O subiéndolo directamente a **Google Colab**.
